## Installation
Get the repo files:
```sh
npm run install-all
```
Copy env file and set the path for `FRONTEND_OUTPUT_PATH`
```sh
cp .env.example .env
```

## Development
```sh
npm run dev
```

## Production
```sh
npm run build && npm start
```
